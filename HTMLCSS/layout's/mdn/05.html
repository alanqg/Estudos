Fluxo Basico do Documento

Elementos básicos de nível de blocos. Os elementos de nível de bloco que ficam um abaixo do outro.

Por padrão pegamos 100$ da largura do nossa elemento pai e temos a mesma altura como o nossao conteúdo filho. Nossa largarua e altura tatais são nosso contúdo + padding + margin + largura/altura da borda

Estamos separados pelas nossas margens. Devido ao colapso das margens, estamos separados pela largura de uma de nossas margens, não de ambas.

Elementos inline como este e este ficam a mesma linha um do outro e nós de texto adjacentes, se houver espaço a mesma linha. O excesso de elementos embutidos irá quebrar em uma nova linha, se possível (como esta contendo texto), ou simplesmente vá para uma nova linha, caso contrário, como esta imagem fará:
